# MAGMA Chat Interface

A ChatGPT-like web interface for MAGMA built with Next.js (Vercel).

## Setup

1. Install dependencies:
```bash
npm install
```

2. Make sure the FastAPI backend is running on `http://localhost:8000`

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Features

- ChatGPT-like chat interface
- Real-time conversation with MAGMA
- Quick action buttons for common queries
- Responsive design with dark theme
- Connects to existing FastAPI backend

## Usage

- Ask questions about stock recommendations
- View your portfolio summary
- Get the latest market news
- Chat naturally with MAGMA about investment decisions
